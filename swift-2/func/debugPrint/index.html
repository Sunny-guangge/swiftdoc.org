func debugPrint<T>(value: T)
func debugPrint<T>(value: T, appendNewline: Bool)

Writes the textual representation of value most suitable for debugging, and an optional newline, into the standard output.

The textual representation is obtained from the value using its protocol conformances, in the following order of preference: CustomDebugStringConvertible, CustomStringConvertible, Streamable. If none of these conformances are found, a default text representation is constructed in an implementation-defined way, based on the type kind and structure.

Do not overload this function for your type. Instead, adopt one of the protocols mentioned above.

appendNewline: Iff true (the default), write a trailing newline.
