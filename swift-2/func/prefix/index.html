func prefix<S : Sliceable>(s: S, _ maxLength: Int) -> S.SubSlice

Returns a slice, up to maxLength in length, containing the initial elements of s.

If maxLength exceeds s.count, the result contains all the elements of s.

Complexity: O(1)+K when S.Index conforms to RandomAccessIndexType and O(N)+K otherwise, where K is the cost of slicing s.
