func split<S : Sliceable, R : BooleanType>(elements: S, maxSplit: Int = default, allowEmptySlices: Bool = default, @noescape isSeparator: (S.Generator.Element) -> R) -> [S.SubSlice]

Returns the result of slicing elements into sub-sequences that don't contain elements satisfying the predicate isSeparator.

maxSplit: The maximum number of slices to return, minus 1. If maxSplit + 1 slices would otherwise be returned, the algorithm stops splitting and returns a suffix of elements.

allowEmptySlices: If true, an empty slice is produced in the result for each pair of consecutive.
