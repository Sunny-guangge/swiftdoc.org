The lazy CollectionType returned by reverse(c) where c is a CollectionType with an Index conforming to BidirectionalIndexType.

Inheritance

CollectionType, SequenceType, _CollectionGeneratorDefaultsType, _CollectionDefaultsType Protocol hierarchy

Associated types

typealias Index = ReverseIndex<Base.Index>

A type that represents a valid position in the collection.

Valid indices consist of the position of every element and a "past the end" position that's not valid for use as a subscript.

typealias Generator = IndexingGenerator<ReverseCollection<Base>>

A type whose instances can produce the elements of this sequence, in order.

Instance Variables

var startIndex: ReverseIndex<Base.Index> { get }

The position of the first element in a non-empty collection.

In an empty collection, startIndex == endIndex.

var endIndex: ReverseIndex<Base.Index> { get }

The collection's "past the end" position.

endIndex is not a valid argument to subscript, and is always reachable from startIndex by zero or more applications of successor().
