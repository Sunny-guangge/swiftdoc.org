An abstract GeneratorType base class over Element elements.

Use this as a Sequence's associated Generator type when you don't want to expose details of the concrete generator, a subclass.

It is an error to create instances of AnyGenerator that are not also instances of an AnyGenerator subclass.

See Also: - struct AnySequence<S : SequenceType> - func anyGenerator<G : GeneratorType>(base: G) -> AnyGenerator<G.Element> - func anyGenerator<Element>(body: ()->Element?) -> AnyGenerator<Element>
