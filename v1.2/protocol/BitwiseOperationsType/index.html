protocol BitwiseOperationsType

A set type with O(1) standard bitwise operators.

Each instance is a subset of ~Self.allZeros

Axioms, where x is an instance of Self:

x | Self.allZeros == x
x ^ Self.allZeros == x
x & Self.allZeros == .allZeros
x & ~Self.allZeros == x
~x == x ^ ~Self.allZeros
