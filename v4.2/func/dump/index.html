func dump<T>(_: T, name: String?, indent: Int, maxDepth: Int, maxItems: Int)

Dumps the given object's contents using its mirror to standard output.

Parameters: value: The value to output to the target stream. name: A label to use when writing the contents of value. When nil is passed, the label is omitted. The default is nil. indent: The number of spaces to use as an indent for each line of the output. The default is 0. maxDepth: The maximum depth to descend when writing the contents of a value that has nested components. The default is Int.max. maxItems: The maximum number of elements for which to write the full contents. The default is Int.max. Returns: The instance passed as value.
