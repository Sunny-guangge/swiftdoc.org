func repeatElement<T>(_: T, count: Int)

Creates a collection containing the specified number of the given element.

The following example creates a Repeated<Int> collection containing five zeroes:

let zeroes = repeatElement(0, count: 5)
for x in zeroes {
    print(x)
}
// 0
// 0
// 0
// 0
// 0

Parameters: element: The element to repeat. count: The number of times to repeat element. Returns: A collection that contains count elements that are all element.
