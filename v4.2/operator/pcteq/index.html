--- layout: "default" title: "Operator: %=" description: "Swift documentation for the '%=' operator." root: "/v4.2" --- operator %= { associativity right precedence assignment }

Declarations

func %=(_: inout Int, rhs: Int)

Divides the first value by the second and stores the remainder in the left-hand-side variable.

The result has the same sign as lhs and is less than rhs.magnitude.

var x = 22
x %= 5
// x == 2

var y = 22
y %= -5
// y == 2

var z = -22
z %= -5
// z == -2

Parameters: lhs: The value to divide. rhs: The value to divide lhs by. rhs must not be zero.
