struct ReverseRandomAccessCollection<Base : Collection where Base.Index : RandomAccessIndex>

A Collection that presents the elements of its Base collection in reverse order.

Note: This type is the result of x.reversed() where x is a collection having random access indices. See Also: ReverseCollection

Associated Types
Index = ReverseRandomAccessIndex<Base.Index>

A type that represents a valid position in the collection.

Valid indices consist of the position of every element and a "past the end" position that's not valid for use as a subscript.

Iterator = IndexingIterator<ReverseRandomAccessCollection<Base>>

A type that provides the sequence's iteration interface and encapsulates its iteration state.
