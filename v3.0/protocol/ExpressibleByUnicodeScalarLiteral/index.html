protocol ExpressibleByUnicodeScalarLiteral

A type that can be initialized with a string literal containing a single Unicode scalar value.

The String, StaticString, Character, and UnicodeScalar types all conform to the ExpressibleByUnicodeScalarLiteral protocol. You can initialize a variable of any of these types using a string literal that holds a single Unicode scalar.

let 単: UnicodeScalar = "単"
print(単)
// Prints "単"

Conforming to ExpressibleByUnicodeScalarLiteral

To add ExpressibleByUnicodeScalarLiteral conformance to your custom type, implement the required initializer.
