protocol NilLiteralConvertible

A type that can be initialized using the nil literal, nil.

nil has a specific meaning in Swift---the absence of a value. Only the Optional type conforms to NilLiteralConvertible. NilLiteralConvertible conformance for types that use nil for other purposes is discouraged.
