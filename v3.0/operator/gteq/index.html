func >= <A : Comparable, B : Comparable, C : Comparable, D : Comparable, E : Comparable, F : Comparable>(_: (A, B, C, D, E, F), rhs: (A, B, C, D, E, F))

Returns a Boolean value indicating whether the first tuple is ordered after or the same as the second in a lexicographical ordering.

Given two tuples (a1, a2, ..., aN) and (b1, b2, ..., bN), the first tuple is after or the same as the second tuple if and only if a1 > b1 or (a1 == b1 and (a2, ..., aN) >= (b2, ..., bN)).

Parameters: lhs: A tuple of Comparable elements. rhs: Another tuple of elements of the same type as lhs.

func >= <T : Comparable>(_: T, rhs: T)

Returns a Boolean value indicating whether the value of the first argument is greater than or equal to that of the second argument.

This is the default implementation of the greater-than-or-equal-to operator (>=) for any type that conforms to Comparable.

Parameters: lhs: A value to compare. rhs: Another value to compare. Returns: true if lhs is greater than or equal to rhs; otherwise, false.
