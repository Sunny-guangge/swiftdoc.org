func fatalError(_: @autoclosure () -> String, file: StaticString, line: UInt)

Unconditionally prints a given message and stops execution.

Parameters: message: The string to print. The default is an empty string. file: The file name to print with message. The default is the file where fatalError(_:file:line:) is called. line: The line number to print along with message. The default is the line number where fatalError(_:file:line:) is called.
